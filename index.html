What we do

Mobile Pro League manages leagues for mobile games and produces highlights, clips and promotional content. We integrate with the TikTok API to let authorized users publish videos directly to their TikTok account.

Technical info (for reviewers)

  • Products used: Login Kit, Content Posting API, Share Kit
  • Scopes: user.info.basic, video.upload, video.publish
  • Data used: only data returned from TikTok API (profile data and video media)

If you need a verification file for TikTok URL-prefix verification, check /.well-known/tiktok-verification.txt.
